# V1.1 LOG

![](../../DOCUMENT/picture/PCB1.png)

## 设计目标

1、首版硬件主要实现了双USB，前面的是ESP32 USB，后面的USB用于调试；

2、实现基础的ESP32外围电路、下载电路、按键周围电路；

3、验证了摇杆封装；

4、为了实现更多的设备链接，增加了NRF2401模块用于实现2.4G射频前端；

5、增加了蜂鸣器；

6、电源部分使用3.7V升压方案，升压到稳定的5V，再降压到3.3V给系统电源使用；

7、可连接一块2.4寸TFT lCD屏幕；

8、确定了PCB形状和遥控器外形3D；
